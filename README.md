# node-template

## Features

- ES6 using Babel
- Linting using ESLint with Airbnb Javascript style guide and Prettier
- ESLint security plugin
- Pre-commit hooks using Lint-Staged with Husky
- Live reloading using Nodemon
- .gitignore for Node

## Installing

```
yarn --dev
```

## Running

```
yarn start
```

## Building

```
yarn build
```
